# Droid Dev Helper 스토어 및 개인정보 정보

## 개요
Droid Dev Helper는 안드로이드 개발자와 기기 제어가 필요한 사용자를 위한 스마트 디바이스 제어 컴패니언 애플리케이션입니다. ADB 및 Scrcpy 연동을 완벽히 내재화하여 복잡한 설정 없이 간편하게 디바이스 미러링, 상태 확인, 시스템 단축키 전송 및 로그캣 분석 작업을 윈도우, 리눅스, 맥 OS 전반에서 동일하게 수행할 수 있습니다.

## 설명
본 애플리케이션은 복잡한 커맨드라인 인터페이스를 사용하지 않고도 안드로이드 개발 및 테스트를 한 눈에 관리할 수 있게 돕는 그래픽 유틸리티입니다. 기기 연결 시 제조사, 모델명, OS 버전 및 해상도를 실시간으로 자동 감지하여 알려주며 백그라운드 스레드 감지 기법을 통해 윈도우에서의 GUI 멈춤(프리징) 없이 부드러운 가동 상태를 선사합니다. 미러링 기능을 사용해 대화면으로 스마트폰을 제어하고, 실시간 로그 스트리밍을 통해 앱 오류 발생 원인을 직관적으로 디버깅할 수 있습니다. 또한 한 번의 클릭만으로 스마트폰의 레이아웃 경계를 시각화하거나 터치 피드백을 노출하고, 개발자 옵션으로 즉시 진입할 수 있어 테스트 및 품질 검증 작업의 리드 타임을 극적으로 단축시킵니다.

## 기능
- 실시간 미러링 및 원격 제어
  기기 연결 후 즉각적으로 고해상도 미러링 창을 띄워 마우스와 키보드로 제어할 수 있습니다.
- 백그라운드 디바이스 자동 감지
  스레드 풀 기반 실시간 모니터링을 통해 시스템 차단 없이 기기의 연결 상태와 시스템 속성을 확인합니다.
- 개발자 도구 간편 단축키
  레이아웃 경계 표기 활성화/비활성화, 터치 피드백 표기, 개발자 옵션 화면으로의 원클릭 진입을 지원합니다.
- 하드웨어 및 단축 키 입력 전송
  기기의 전원 버튼, 홈 버튼, 뒤로 가기 버튼, 앱 전환 버튼 입력을 프로그램 GUI에서 직접 제어할 수 있습니다.
- 텍스트 입력 및 패키지 데이터 초기화
  입력 칸에 한글/영어 텍스트를 기입하여 폰으로 일괄 전송하거나 특정 패키지 명의 앱 데이터를 손쉽게 초기화합니다.
- 실시간 로그캣 스트리밍 및 캡처
  스마트폰 내부의 실시간 디버그 로그를 스크롤뷰로 파이닝하고 간편하게 바탕화면 경로로 스크린샷 이미지를 저장합니다.
